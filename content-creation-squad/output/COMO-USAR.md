# 🎨 Como Usar a Integração Figma

## 🚀 Modo Rápido (3 passos)

### 1. Ver Preview no Navegador
```bash
# O arquivo já deve estar aberto, mas caso não esteja:
open preview-stories.html
```

### 2. Copiar Plugin para Figma Desktop

**a) Abrir arquivo:**
```bash
# Mac:
open figma-plugin-ready.js

# Ou abrir manualmente em qualquer editor de texto
```

**b) Copiar TODO o conteúdo** (Cmd+A, Cmd+C)

**c) Colar no Figma Desktop:**
1. Abrir Figma Desktop App
2. Menu → Plugins → Development → New Plugin...
3. Escolher template "Empty"
4. Nomear: "Stories Creator"
5. **Colar o código copiado**
6. Salvar (Cmd+S)

### 3. Executar Plugin
1. Menu → Plugins → Development → Stories Creator
2. Aguardar ~30 segundos
3. ✅ **7 Stories criados!**

## 📤 Exportar para Instagram

**No Figma Desktop:**
1. Selecionar todos os 7 frames (Cmd+A)
2. Botão direito → Export...
3. Configurar:
   - Format: **PNG**
   - Scale: **2x** (melhor qualidade)
4. Clicar em **Export**
5. Salvar na pasta de Downloads

**Transferir para celular:**
- AirDrop (Mac → iPhone)
- Google Drive / Dropbox
- Cabo USB

**Postar no Instagram:**
1. Instagram → Criar Story
2. Upload das 7 imagens em sequência
3. Adicionar stickers interativos:
   - Story 1: Caixa de pergunta
   - Story 2: Emoji slider  
   - Story 7: Enquete
4. Publicar!

## 🆘 Problemas?

### Plugin não roda
- ✅ Verificar se está usando Figma **Desktop** (não web)
- ✅ Copiar TODO o código (incluindo primeira e última linha)
- ✅ Salvar antes de executar (Cmd+S)

### Fontes não aparecem
- O plugin usa "Inter" (fonte padrão do Figma)
- Se não aparecer, mude para outra fonte:
  - Encontre no código: `{ family: "Inter", style: "Bold" }`
  - Substitua por: `{ family: "Roboto", style: "Bold" }`

### Figma Desktop não instalado
- Download: https://www.figma.com/downloads/
- É NECESSÁRIO o app Desktop (não funciona no browser)
