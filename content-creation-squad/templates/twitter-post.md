# Template: Twitter-Style Post

## Especificações Técnicas

```yaml
platform: Instagram, LinkedIn, Facebook
style: Twitter/X Screenshot
format: 1080x1350px (4:5) ou 1080x1080px (1:1)
file_format: PNG (melhor qualidade)
background: Branco (#FFFFFF) ou Cinza claro (#F7F9F9)
```

## Elementos Visuais Obrigatórios

### 1. Foto de Perfil
- **Posição**: Top-left, ~20-40px da borda
- **Tamanho**: 48x48px (circular)
- **Qualidade**: Alta resolução, profissional
- **Formato**: Circular com borda opcional

### 2. Nome do Usuário
- **Posição**: Ao lado da foto de perfil
- **Fonte**: Helvetica Neue Bold ou similar
- **Tamanho**: 20px
- **Cor**: #0F1419 (preto Twitter)
- **Adicional**: Emoji opcional ao lado (💰🚀🧠)

### 3. Username (@usuario)
- **Posição**: Abaixo do nome
- **Fonte**: Helvetica Neue Regular
- **Tamanho**: 16px
- **Cor**: #536471 (cinza Twitter)
- **Formato**: @username

### 4. Texto do Post
- **Posição**: Abaixo do header (~100px do topo)
- **Fonte**: Helvetica Neue Regular
- **Tamanho**: 18-22px
- **Cor**: #0F1419
- **Linha de altura**: 1.4-1.5
- **Máximo**: 280 caracteres (ideal)
- **Quebras**: 2-4 linhas curtas melhor que bloco

## Estruturas de Copy Comprovadas

### Estrutura 1: TEASER + CTA
```
[Hook impactante]

[Criar expectativa]

[CTA com palavra-chave]
```

**Exemplo**:
```
Isso é um teste. Daqui 30 dias eu volto com os resultados.

Enquanto isso...

Comente "funil" se quiser copiar e colar a estrutura
que gera mais de 700 reuniões por mês em meu negócio 👆🏻
```

### Estrutura 2: RESULTADO + SISTEMA
```
[Resultado específico]

[Sistema resumido]

[CTA]
```

**Exemplo**:
```
Sistema de IA criou meu último post em 10 minutos.

Copy otimizado + estratégia de engajamento +
gatilhos de venda, tudo automatizado.

Comente "SISTEMA" se quiser ter acesso
```

### Estrutura 3: TRANSFORMAÇÃO
```
De [ANTES]
Para [DEPOIS]

[Como]

[CTA]
```

**Exemplo**:
```
De 3 horas criando 1 post
Para 10 minutos criando posts que vendem

O segredo? Sistema de IA treinado com as melhores estratégias.

Salva esse post
```

### Estrutura 4: LISTA DE VALOR
```
[NÚMERO] [COISAS] que [RESULTADO]:

1. [Item 1]
2. [Item 2]
3. [Item 3]

[CTA]
```

**Exemplo**:
```
3 gatilhos que uso todo dia pra vender:

1. Prova social - mostre quem já comprou
2. Urgência - oferta com prazo real
3. Garantia - reverta o risco do cliente

Qual você usa?
```

### Estrutura 5: POLÊMICA
```
Opinião impopular:

[Declaração controversa]

[Argumento breve]

[CTA debate]
```

**Exemplo**:
```
Opinião impopular:

Criar conteúdo "autêntico" é desculpa pra preguiça.

Copy que converte > "ser você mesmo"

Concorda?
```

## Paleta de Cores (Twitter Official)

### Cores Principais
- **Fundo**: #FFFFFF (branco puro)
- **Fundo alternativo**: #F7F9F9 (cinza clarinho)
- **Texto principal**: #0F1419 (quase preto)
- **Texto secundário**: #536471 (cinza médio)

### Cores Opcionais (emojis/destaques)
- Use emojis nativos (não cores customizadas)
- Máximo 1-2 emojis por post

## Tipografia

### Fontes Recomendadas
**Opção 1** (mais fiel):
- Helvetica Neue

**Opção 2** (alternativa gratuita):
- Inter
- SF Pro Display
- Roboto

### Tamanhos
- **Nome**: 20px, Bold (700)
- **Username**: 16px, Regular (400)
- **Post**: 18-22px, Regular (400)

### Espaçamento
- **Line-height**: 1.4-1.5 (entre linhas)
- **Padding**: 40px (geral)
- **Gap header-post**: 20-25px

## Dicas Pro

### Para Máximo Engajamento
1. **Primeira linha**: Decide 70% do engajamento
2. **Quebras curtas**: Mais fácil de ler
3. **Números específicos**: "700 reuniões" > "muitas reuniões"
4. **CTA simples**: "Comente X" melhor que "me manda DM"
5. **Emoji pontual**: 1-2 no máximo

### Para Parecer Autêntico
1. Use sua foto real
2. Mantenha consistência de nome/username
3. Não exagere em edição
4. Copy conversacional (como você falaria)
5. Evite muito formalismo

### Para Instagram/LinkedIn
1. Caption adicional explicando contexto
2. Postar em horário de pico
3. Engajar nos primeiros comentários
4. Repostar stories (aumenta alcance)
