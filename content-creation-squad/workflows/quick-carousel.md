# Workflow: Quick Carousel (5 minutos)

**Duração**: 5 minutos
**Objetivo**: Criar carrossel funcional rapidamente
**Diferença do workflow completo**: Menos customização, mais velocidade
**Quando usar**: Produção em volume, testes rápidos, conteúdo oportunista

---

## ⚡ Overview

Este workflow sacrifica customização profunda em favor de velocidade. Usa templates prontos e decisões pré-definidas.

**Comparação**:
- **Workflow Completo** (`carousel-full-flow.md`): 10-15 min, 100% customizado
- **Quick Carousel**: 5 min, 80% de qualidade, templates prontos

---

## 🎯 PASSO 1: Inputs Rápidos (30 segundos)

Apenas 3 perguntas obrigatórias:

### 1. Tema em 1 frase
> Exemplo: "5 erros de copy que matam conversão"

### 2. Tipo de carrossel
- [ ] **Educacional** (ensinar algo)
- [ ] **Vendas** (vender produto/serviço)
- [ ] **Viral** (gerar shares e discussão)

### 3. Plataforma
- [ ] Instagram
- [ ] LinkedIn
- [ ] Ambas (usa padrão profissional)

**Opcional** (se tiver tempo):
- CTA preferido: [comentar / salvar / compartilhar / clicar link]

---

## 📋 PASSO 2: Escolher Fórmula Automática (30 segundos)

Baseado no tipo escolhido, use a fórmula padrão:

### Se EDUCACIONAL → Fórmula "Problema-Solução" (8 slides)
```
1. Hook (problema específico)
2. Agitação (por que isso importa)
3-7. Soluções (uma por slide)
8. Resumo
9. CTA educacional
10. Closer
```

**Awareness recomendado**: Solution Aware
**Hook type**: Number ("X erros/maneiras/táticas")

---

### Se VENDAS → Fórmula "Venda Direta" (7 slides)
```
1. Hook (promessa grande)
2. Problema atual
3. Solução (seu produto)
4. Como funciona
5. Resultados/prova
6. Oferta/preço
7. CTA de conversão
```

**Awareness recomendado**: Product Aware ou Most Aware
**Hook type**: Command ou Promise

---

### Se VIRAL → Fórmula "Controvérsia" (6 slides)
```
1. Hook polêmico
2. Por que opinião impopular
3. Argumento 1
4. Argumento 2
5. Argumento 3
6. CTA de discussão
```

**Awareness recomendado**: Problem Aware
**Hook type**: Controversial

---

## ✍️ PASSO 3: Hook Rápido (1 minuto)

### Processo Express:

1. Abrir `templates/hook-library.md`
2. Ir para awareness stage recomendado (acima)
3. Escolher formato recomendado
4. Pegar primeiro template da lista
5. Substituir [PLACEHOLDERS] com seu tema

### Exemplo:

**Tema**: "5 erros de copy que matam conversão"
**Tipo**: Educacional → Hook type: Number
**Template escolhido**: "[X] erros que [CONSEQUÊNCIA]"

**Hook final**:
```
5 erros de copy
que estão matando
sua conversão

(e como corrigir cada um)
```

**Tempo**: 1 minuto máximo. Primeira opção é boa o suficiente.

---

## 📝 PASSO 4: Preencher Estrutura (2 minutos)

Use a fórmula escolhida como template.

### Para EDUCACIONAL (exemplo):

**SLIDE 1 - HOOK**
> [Hook criado no passo 3]

**SLIDE 2 - AGITAÇÃO**
> [Tema] está custando [R$ / clientes / tempo]

Exemplo:
```
Seu copy está deixando
R$ 50 mil na mesa
todo mês.

E você nem percebe.
```

**SLIDES 3-7 - SOLUÇÕES**
> ERRO/TÁTICA #[N]
> [Nome do erro]
> [1 linha de explicação]

Exemplo Slide 3:
```
ERRO #1
Falar de features
ao invés de benefícios

Ninguém quer "IA integrada"
Querem "economizar 5h/dia"
```

**SLIDE 8 - RESUMO**
> Recapitulando:
> ✓ [Solução 1]
> ✓ [Solução 2]
> ✓ [Solução 3]

**SLIDE 9 - CTA**
> [CTA da biblioteca]

Use `templates/cta-library.md`:
- Educacional → "Salva para consultar antes da próxima copy"
- Vendas → "DM '[PALAVRA]' para receber [OFERTA]"
- Viral → "Comenta se concorda ou discorda"

**SLIDE 10 - CLOSER**
> Qual você vai aplicar primeiro?
>
> Comenta aqui 👇
>
> [@seu_usuario]

---

## 📱 PASSO 5: Caption Express (1 minuto)

### Estrutura Rápida:

```
[Hook expandido em 1-2 frases]

[1 parágrafo contexto - 2-3 linhas]

[CTA + benefício]
```

### Exemplo Completo:

```
5 erros de copy que estão matando sua conversão (e você nem percebe).

A maioria dos copywriters foca nas palavras erradas. Resultado? Copy bonita que não converte. No carrossel acima, mostro os 5 erros mais comuns e como corrigir cada um em menos de 5 minutos.

Salva este checklist e revisa antes de escrever sua próxima copy.

#copywriting #marketing #vendas #conversão #digitalmarketing
```

**Tempo**: 1 minuto. Não perfeccionar, só fazer funcionar.

---

## ✅ PASSO 6: Review Express (30 segundos)

Checklist rápido (apenas YES/NO, sem perfeccionismo):

- [ ] Hook para o scroll?
- [ ] Máximo 2-3 linhas por slide?
- [ ] CTA está claro?
- [ ] Zero typos óbvios?

Se 4/4 = YES → **PUBLICAR**

Se algum NO → Corrigir apenas aquilo (30 seg extras)

## 📤 OUTPUT FINAL

### Documento Simples:

```markdown
# CARROSSEL: [Título]

## SLIDES

### 1 - HOOK
[texto]

### 2 - AGITAÇÃO
[texto]

### 3-7 - CONTEÚDO
[texto de cada]

### 8 - RESUMO
[texto]

### 9 - CTA
[texto]

### 10 - CLOSER
[texto]

## CAPTION
[caption completa]

## NOTAS
- Plataforma: [IG/LI]
- Paleta: [cores básicas]
- Fonte: Montserrat/Roboto
```

**Sem**:
- Planejamento visual detalhado
- Múltiplas opções de hook
- Otimização de vendas complexa
- Localização BR específica

**Suficiente para**:
- Passar para designer
- Publicar com template visual pronto
- Testar conceito rapidamente

## 🔄 Uso com Batch Production

Este workflow é IDEAL para `batch-production.md`:

```
Loop 10x:
  1. quick-carousel.md (5 min)
  2. Salvar output/carrossel-[N].md
  3. Next tema

Total: 50 minutos = 10 carrosséis prontos
```

## 💡 Dicas para Máxima Velocidade

### 1. Tenha banco de temas
Liste 20-30 temas de antemão:
```
- 5 erros de [tema]
- Como [resultado] em [prazo]
- [número] táticas de [área]
- O método para [objetivo]
```

### 2. Use Notion/planilha
Template com campos:
- Tema: _______
- Tipo: [dropdown]
- Plataforma: [dropdown]
- Hook: _______
- Slides 3-7: _______

Preenche → copia → formata → pronto

### 3. Reutilize estruturas
Se um carrossel performou bem:
- Mantenha mesma fórmula
- Mude apenas o tema
- 3 minutos para criar variação

### 4. Batch por tema
Criar 5 carrosséis sobre o mesmo tema:
- Reaproveita research
- Variações são mais rápidas
- Teste de ângulos diferentes

## ⚠️ Limitações do Quick Carousel

**Não use para**:
- Lançamentos importantes
- Vendas de produtos premium
- Quando qualidade > velocidade
- Primeira impressão crítica

**Use para**:
- Conteúdo recorrente/semanal
- Testes de conceito
- Produção em volume
- Oportunidades rápidas (trending topics)

---

## 📊 Performance Esperada

**Quick Carousel bem executado**:
- 70-80% da performance de um carrossel completo
- 3x mais rápido de criar
- Ótimo para volume e consistência

**Benchmark**:
- Se carrossel completo gera 1000 saves
- Quick carousel deve gerar 700-800 saves
- Mas você cria 3x mais em mesmo tempo

**ROI**: Melhor para manter presença constante

---

## 🔄 Quando Upgradear para Workflow Completo

Sinais que um tema merece workflow completo:

1. **Performance do quick carousel foi excepcional**
   - Refaça versão completa e republique

2. **Tema está alinhado com lançamento**
   - Use workflow completo desde o início

3. **Você tem tempo extra**
   - Pegue 2-3 quick carousels que performaram
   - Refaça com workflow completo
   - Republique como "versão 2.0"
