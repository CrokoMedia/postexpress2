# Workflow: Quick Audit

**ID**: quick-audit
**Duration**: 10-15 min
**Output**: Score card rápido + Top 5 insights

## Descrição
Versão enxuta da auditoria completa. Ideal para análise rápida antes de uma
reunião, para triagem de concorrentes ou primeira impressão de uma conta.

## Etapas

### 1. INPUT RÁPIDO (2 min)
```
Fornecer:
  - @username OU 10-20 posts principais
  - Objetivo: o que mais importa analisar?
```

### 2. ANÁLISE SIMULTÂNEA (8 min)
```
Todos os 5 auditores em paralelo analisam amostra de 10 posts:

Kahneman   → Engajamento real vs fake, padrão emocional
Schwartz   → Hook quality + Awareness Stage distribuição
Hormozi    → CTA score + Value Equation rápida
Cagan      → Outcomes identificados na amostra
Paul Graham→ 1 anomalia principal + 1 oportunidade
```

### 3. CONSOLIDAÇÃO (3 min)
```
Output:
  ✓ Score Geral (0-100)
  ✓ Top 3 forças
  ✓ Top 3 problemas
  ✓ 1 Quick Win imediato
  ✓ 1 Strategic insight
```

## Quando Usar
- Primeira análise de uma conta nova
- Triagem rápida de lista de concorrentes
- Check-in mensal de uma conta monitorada
- Validação rápida antes de proposta comercial
