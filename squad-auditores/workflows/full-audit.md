# Workflow: Full Audit

**ID**: full-audit
**Duration**: 30-60 min
**Output**: Relatório completo de auditoria

## Descrição
Auditoria profunda e completa de uma conta de rede social com dados de scraping.
Todos os 5 auditores são ativados em sequência para análise multidimensional.

## Etapas

### 1. INGESTÃO (5 min)
```
audit-lead recebe dados:
  - Processa JSON/CSV do scraper
  - Organiza posts por data
  - Extrai métricas principais
  - Identifica amostra representativa
```

### 2. KAHNEMAN: Comportamento (10 min)
```
Análise:
  ✓ Engajamento System 1 (curtidas, views) vs System 2 (salvamentos, compartilhamentos)
  ✓ Loss Aversion indicators
  ✓ Heurísticas cognitivas ativadas
  ✓ Ruído vs Viés nas métricas
  ✓ Padrões de retenção de audiência

Output: Behavioral Analysis Report
```

### 3. SCHWARTZ: Copy (10 min)
```
Análise:
  ✓ Awareness Stage de cada post (amostra)
  ✓ Hook Score médio
  ✓ Market Sophistication Level
  ✓ Mapeamento de desejos canalizados
  ✓ Especificidade e credibilidade

Output: Copy Audit Report
```

### 4. HORMOZI: Oferta (8 min)
```
Análise:
  ✓ Value Equation de CTAs identificados
  ✓ Grand Slam Offer check
  ✓ Vocabulário auditado
  ✓ Prova social score
  ✓ Risk reversal presente?

Output: Offer Audit Report
```

### 5. CAGAN: Métricas (7 min)
```
Análise:
  ✓ Outcomes vs Outputs breakdown
  ✓ Four Risks Assessment
  ✓ Discovery vs Delivery classification
  ✓ North Star Metric identification
  ✓ Trend analysis

Output: Metrics Audit Report
```

### 6. PAUL GRAHAM: Anomalias (5 min)
```
Análise:
  ✓ Top anomalias (positivas e negativas)
  ✓ Padrões contraintuitivos
  ✓ Gaps de oportunidade
  ✓ Hipóteses para testar

Output: Anomaly Detection Report
```

### 7. CONSOLIDAÇÃO (5 min)
```
audit-lead sintetiza:
  ✓ Score Card Geral (0-100)
  ✓ Score por dimensão
  ✓ Top 3 forças
  ✓ Top 3 fraquezas críticas
  ✓ 5 recomendações priorizadas por ROI
  ✓ Quick Wins (ações imediatas)
  ✓ Strategic Moves (90 dias)
```
