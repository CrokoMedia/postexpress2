🚨 Problemas Identificados

🔴 CRÍTICO: Duplicação de Modal

Situação: 2 implementações de modal coexistindo no sistema:

  • atoms/dialog.tsx - Radix UI (acessível, animado, robusto)
  • molecules/modal.tsx - Custom HTML (básico, sem a11y)

Impacto: Manutenção duplicada, inconsistência visual, falta de acessibilidade ARIA

Solução: Migrar todos modais para Dialog atom (Radix) e deletar modal.tsx

🟡 MÉDIO: Cores Hardcoded

Situação: 7 arquivos usam purple/violet/indigo ao invés de primary-*

  • app/vendas/_components/ (8 componentes)
  • components/organisms/auditor-section.tsx
  • components/twitter/twitter-experts-section.tsx
  • app/not-found.tsx
  • app/login/page.tsx

Impacto: Rebrand futuro exigiria editar 7+ arquivos manualmente

Solução: Buscar/substituir global: purple-* → primary-*

🟢 BAIXO: Escalas de Cores Incompletas

Situação: success, warning, error, info só têm 2-3 shades (faltam 50-900)

Impacto: Limitação na criação de variantes (hover, active, disabled)

Solução: Gerar escalas completas usando uicolors.app
